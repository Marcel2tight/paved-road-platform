# Backstage Operational Plugins Roadmap

## Objective

The Paved Road Platform uses Backstage as its Internal Developer Portal.

This roadmap documents current capabilities and planned integrations to continuously improve developer experience, platform visibility, governance, and operational excellence.

---

# Currently Implemented

## Software Templates

Status: ✅ Implemented

Capabilities:

- Cloud Run self-service provisioning
- Infrastructure scaffolding
- Pull Request generation
- Standardized deployment workflows

---

## Service Catalog

Status: ✅ Implemented

Capabilities:

- Platform component registration
- System registration
- Ownership metadata
- Lifecycle metadata

---

## TechDocs

Status: ✅ Implemented

Capabilities:

- Platform documentation
- Golden Paths
- Developer onboarding

# Planned Plugins

Status: ⏳ Planned

- View Kubernetes workloads
- Inspect pods, deployments, and services
- Troubleshoot platform workloads

Status: ⏳ Planned
