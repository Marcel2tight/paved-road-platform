# Grafana Dashboards

This directory stores exported Grafana dashboard JSON files for the Paved Road Platform.

## Planned Dashboards

| File | Dashboard |
|---|---|
| `platform-overview.json` | Paved Road Platform Overview |
| `cloud-run-operations.json` | Cloud Run Operations |
| `reliability-slos.json` | Reliability and SLOs |
| `developer-experience.json` | Developer Experience |

Dashboards are initially built using the Google Cloud Monitoring data source in Grafana Cloud.

Future dashboards may include Managed Prometheus, BigQuery billing export, and Vertex AI metrics.
